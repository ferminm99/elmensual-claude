# Cómo usar este sistema (léelo una vez)

> Este sistema es el "cerebro" de El Mensual LP. Sirve para que no tengas que explicarme el contexto cada vez que quieras un guion, una campaña o una idea.

---

## Cuándo y cómo me pasás los archivos
- **Al empezar una conversación de contenido**, decime: *"Trabajemos con el sistema de El Mensual"*. Si no los tengo cargados, pasame la carpeta `marca-mensual-lp/` o el/los archivos relevantes.
- Para algo puntual, alcanza con el `00-resumen-ejecutivo.md` + el archivo del cliente al que le apuntás.

## Cómo me pedís un guion nuevo (la forma ideal)
Decímelo con estos 3 datos y te lo devuelvo afinado:
1. **A qué cliente** (campo / interior / nuevo).
2. **Qué etapa del embudo** (🧊 atraer / 🌤️ que conozcan / 🔥 que compren).
3. **Producto o ángulo** (ej. poplín tiro alto, comparación con jean, testimonio).

> Ejemplo: *"Guion de reel para cliente nuevo, etapa fría, mostrando el calce de la poplín tiro alto."*
> O más corto: *"Dame 3 ganchos para vender corderoy en invierno al cliente nuevo."*
> Para carruseles: *"Armame un carrusel comparativo de grafa para cliente nuevo."*

## Qué archivos actualizás VOS con el tiempo
Estos son vivos, mantenelos al día:
- **`02-clientes/cliente-nuevo.md`** → cuando preguntes a clientes reales POR QUÉ compraron, pegá sus frases textuales ahí. Es lo más valioso.
- **`03-contenido/banco-de-angulos.md`** → sumá ideas nuevas y marcá qué formatos funcionaron.
- **`04-referencias/competencia.md`** → agregá las marcas de referencia que mires.
- **`01-identidad/producto.md`** → actualizá precios cuando cambien, y resolvé los `[CONFIRMAR]`.
